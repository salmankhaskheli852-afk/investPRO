# **App Name**: InvesPro

## Core Features:

- Agent Panel: Dashboard for agents to manage user interactions and investment plans.
- Admin Panel: Centralized control panel for managing agents, users, investment plans, and wallet details. This feature includes the ability to edit and add investment plans with names, prices, daily income percentages, total income, and image URLs.
- User Panel: Dashboard for users to view investment plans, manage their wallets, and interact with agents.
- Google Authentication: Secure user authentication via Google OAuth.
- Investment Plans: Display investment plans as designed cards with details such as name, price, percentage daily income, total income, and an image sourced from a URL.
- Wallet Management: Functionality for users to deposit and withdraw funds. Deposits use admin-provided wallet details (number, name, wallet name), while withdrawals require the user to enter the amount, wallet number, and wallet name.
- Inter-Panel Communication: Enable seamless communication and data flow between agent, admin, and user panels.

## Style Guidelines:

- Primary color: Vivid blue (#3498DB) to convey trust and reliability in financial dealings.
- Background color: Light gray (#ECF0F1), providing a neutral backdrop that emphasizes the colorful plan cards.
- Accent color: Energetic orange (#E67E22) for calls to action and important notifications.
- Body and headline font: 'PT Sans', a versatile sans-serif providing a balance of modernity and approachability.
- Use financial and security-themed icons across all panels to reinforce trust and professionalism.
- Employ a clean, card-based layout for investment plans and wallet details, ensuring information is easily digestible and visually appealing.
- Incorporate subtle animations when investment plans are purchased or wallet transactions are made, enhancing user engagement.
